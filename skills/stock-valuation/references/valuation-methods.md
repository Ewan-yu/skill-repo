# 估值方法详解

> 基于《价值投资入门与实践》的估值方法体系

---

## 一、相对估值法概述

相对估值法之所以流行，不是因为它最本质，而是因为它最容易上手。问题也恰恰在这里：越容易上手，越容易被误用。很多投资者不是不会估值，而是把本来适用于一类公司的方法，硬套到另一类公司上。

---

## 二、PE估值：最常见，也最容易被误用

### 2.1 公式

```
PE = 股价 / 每股收益 EPS = 总市值 / 归母净利润
```

**常见口径**：

| 口径 | 公式 | 用途 |
|------|------|------|
| **静态PE** | 当前股价 / 上一年EPS | 历史对比 |
| **TTM PE** | 当前股价 / 最近四个季度EPS | 最常用 |
| **预估PE** | 当前股价 / 未来一年预测EPS | 投资决策首选 |

> **叶城观点**："我在实战里最常用的是预估PE。原因很简单：投资看未来，不看过去。"

### 2.2 用PE算目标市值和目标价

```
目标市值 = 预测净利润 × 目标PE
目标价   = 预测EPS × 目标PE
```

### 2.3 PE的倒数：盈利收益率

```
PE的倒数 = 盈利收益率

例如：
20倍 PE ≈ 1/20 = 5% 的盈利收益率
25倍 PE ≈ 1/25 = 4% 的盈利收益率
```

**参考意义**：如果无风险收益率、理财收益率在4%左右，那么25倍PE在静态意义上就不算离谱。但这只是一个粗的参照，真正决定PE的，仍然是成长性和确定性。

### 2.4 PE适用场景

**适合用PE的公司**：
1. 利润较稳定的消费股
2. 商业模式成熟、盈利质量较高的价值股
3. 部分成长股的短期估值比较

**不适合用PE的公司**：
1. 周期股的利润高点阶段
2. 高杠杆金融股
3. 当前亏损、利润被前置投入压低的公司

### 2.5 合理PE怎么给

**四类参考锚**：
1. 无风险收益率
2. 市场平均估值
3. 行业平均估值
4. 公司自身历史估值区间

> **叶城观点**："这些锚都能看，但都不能机械用。真正决定PE的，还是成长性、生命周期、商业模式和盈利确定性。"

### 2.6 用PE前的清洗步骤

**第一步**：分清是静态、TTM还是预估PE

**第二步**：剔除非经常性损益
- 出售资产带来的利润，不能当成常态盈利
- 一次性关厂、减值、补贴等，也要单独看待

**警告**：否则看上去"只有12倍PE"的公司，可能只是当年利润被一次性项目放大了。

### 2.7 PE估值判断标准

| PE值 | 判断 | 适用 |
|------|------|------|
| <15倍 | 明显低估 | 成熟行业 |
| 15-25倍 | 合理偏低 | 稳定增长 |
| 25-40倍 | 合理 | 高质量公司 |
| 40-60倍 | 偏高 | 高成长 |
| >60倍 | 高估 | 需验证 |

## 三、PB估值：净资产口径的核心工具

### 3.1 公式

```
PB = 股价 / 每股净资产 BPS = 总市值 / 归母净资产
```

### 3.2 用PB算目标价

```
目标市值 = 目标PB × 预测归母净资产
目标价   = 目标PB × 预测每股净资产 BPS
```

### 3.3 PB适用场景

**最适合用PB的公司**：
1. 银行
2. 券商
3. 保险（更常用变体PEV）
4. 其他高杠杆、利润容易失真的企业

### 3.4 银行股PB估值：招商银行案例

**测算思路**：
1. 先预测未来每股净资产
2. 再给目标PB

**示例**：
```
若预测下一年 BPS = 23元
若给予 1.1倍 PB
则目标价 ≈ 23 × 1.1 = 25.3元
```

> **叶城观点**："这只是测算框架，不是投资建议。真正的关键，是先判断ROE、资产质量和成长性是否配得上这个PB。"

### 3.5 银行特有公式

```
合理PB ≈ ROE × 1（理论值）
实际PB = ROE × 0.8~1.2（调整后）
```

**调整系数说明**：
- 0.8：资产质量一般、成长性一般
- 1.0：资产质量良好、成长性稳定
- 1.2：资产质量优秀、成长性好（如招商银行）

### 3.6 PB估值判断标准

| PB值 | 判断 | 适用 |
|------|------|------|
| <0.8倍 | 明显低估 | 银行、周期 |
| 0.8-1.2倍 | 合理 | 银行、周期 |
| 1.2-2倍 | 偏高 | 优质银行 |
| 2-3倍 | 高估 | 非银金融 |
| >3倍 | 明显高估 | 特殊情况 |

## 四、PEG估值：成长股最实用的估值方法

### 4.1 公式

```
PEG = PE / 未来3-5年净利润复合增速
```

**注意**：增速这里通常按百分数口径使用。比如利润复合增速30%，若PEG=1，则对应合理PE约为30倍，而不是0.3倍。

### 4.2 PEG的实战推导

```
当 PEG = 1 时：
目标PE ≈ 未来3-5年净利润复合增速（百分数口径）

目标市值 = 预测净利润 × 目标PE
目标价   = 预测EPS × 目标PE
```

### 4.3 PEG的标准使用步骤

1. **先预测未来3-5年净利润复合增速**
2. **先把PEG=1作为基础合理值**
3. **再根据行业空间和竞争格局，对目标PE做折价或溢价**

**溢价/折价原则**：
- 确定性特别高：可以适度高于1
- 行业空间一般、竞争激烈：就要低于1

### 4.4 PEG实战案例：宝信软件

**测算步骤**：

1. 用分析师预测EPS，推导未来几年利润增速
2. 根据未来三年复合增速，给PEG基准PE（25-30倍）
3. 结合IDC资源属性、竞争格局和确定性，决定溢价或折价

**具体计算**：
```
2018预估EPS = 0.73元
2019预估EPS = 0.94元

2019增速 ≈ 0.94 / 0.73 - 1 = 28.8%
```

若判断未来三年复合增速30%：
```
PEG = 1 时，目标PE ≈ 30倍
```

考虑IDC资源属性、土地电力指标等优势，给10%溢价：
```
目标价 ≈ 30 × 0.94 × 1.1 = 31.02元
```

> **叶城观点**："PEG不是只算一个比值，而是先用它锚定合理PE，再做结构性修正。"

### 4.5 PEG判断标准

| PEG值 | 判断 | 操作 |
|-------|------|------|
| <0.8 | 深度低估 | 买入 |
| 0.8-1 | 低估 | 买入 |
| 1-1.5 | 合理 | 持有 |
| 1.5-2 | 偏高 | 观望 |
| >2 | 高估 | 卖出 |

## 五、PS估值：利润不适合看时，用销售额做锚

### 5.1 公式

```
PS = 股价 / 每股销售额 = 总市值 / 营业收入
```

### 5.2 用PS算目标市值和目标价

```
目标市值 = 目标PS × 预测营业收入
目标价   = 目标市值 / 总股本
```

### 5.3 PS适用场景

**PS不是给亏损公司遮羞用的**。它真正适合的是：
1. 当前利润被前置投入压低
2. 未来净利率改善路径清晰
3. 客户续费率高、商业模式稳定

**最典型的就是云计算SaaS**。

### 5.4 为什么云SaaS可以用PS

**核心逻辑**：优秀云计算企业在成熟后，净利率有机会做到30%左右。所以，10倍PS最终可能对应30多倍PE。

**关键**：PS背后必须有未来利润率的支撑，否则高PS就没有意义。

### 5.5 PS估值判断标准

| PS值 | 判断 | 适用 |
|------|------|------|
| <5倍 | 低估 | 成熟企业 |
| 5-10倍 | 合理 | 稳定增长 |
| 10-20倍 | 偏高 | 高成长 |
| >20倍 | 高估 | 需验证 |

## 六、PEV估值：保险行业专用的净资产变体

### 6.1 公式

```
PEV = 股价 / 每股内含价值 EV = 总市值 / 内含价值 EV
```

### 6.2 用PEV算目标价

```
目标市值 = 目标PEV × 预测内含价值 EV
目标价   = 目标PEV × 预测每股内含价值
```

### 6.3 中国平安案例：如何推目标价

**测算步骤**：
1. 先预测每股内含价值
2. 再根据新业务价值增速和公司质量给PEV

**示例**：
```
若预测下一年每股EV为76元
若给1.0倍PEV
则目标价 ≈ 76元
```

### 6.4 PEV估值判断标准

| PEV值 | 判断 | 适用 |
|-------|------|------|
| <0.8倍 | 明显低估 | 保险股 |
| 0.8-1.2倍 | 合理 | 保险股 |
| 1.2-1.5倍 | 偏高 | 优质保险 |
| >1.5倍 | 高估 | 需验证 |

## 七、相对估值的未来测算标准动作

无论用PE、PB、PEG、PS还是PEV，都应按下面顺序走：

1. **先确认企业适用哪种方法**
2. **预测未来1-3年关键财务指标**
3. **给目标估值倍数**
4. **算出目标市值**
5. **再算出目标价**
6. **最后做悲观、中性、乐观三套情景**

### 五种方法的落地模板

```
PE  ：预测EPS   × 目标PE  = 目标价
PB  ：预测BPS   × 目标PB  = 目标价
PEG ：预测EPS   ×（增速 × PEG修正后倍数）= 目标价
PS  ：预测收入   × 目标PS  = 目标市值，再除总股本
PEV ：预测每股EV × 目标PEV = 目标价
```

## 八、如何防止相对估值被误用

### 8.1 三件事必须做

1. **至少用两种方法交叉验证**
2. **必须先判断行业和公司所处阶段**
3. **绝不只因为"同行是40倍"就给这家公司40倍**

> **叶城观点**："真正可靠的估值，最后往往是多种方法算出来的结果大致接近，而不是永远依赖单一倍数。"

### 8.2 戴维斯双击和双杀

```
股价涨幅 ≈ 业绩增长 + 估值抬升
股价跌幅 ≈ 业绩下滑 + 估值压缩
```

**含义**：相对估值不只是"现在几倍"，更是在判断：未来市场会不会给更高倍数，还是会把倍数打下来。

## 九、估值陷阱识别

### 9.1 低PE陷阱

**情况**：周期股在盈利高点时PE很低

**案例**：钢铁股2018年
- 盈利高点，PE 5倍
- 看起来便宜
- 实际上是陷阱

**应对**：
- 周期股不看PE
- 看PB和市值
- 把握周期节奏

### 9.2 伪成长陷阱

**情况**：过去高增长，PE合理
- 但增长不可持续
- 买入后增速下滑
- 估值和盈利双杀（戴维斯双杀）

**应对**：
- 分析增长的可持续性
- 看行业空间
- 看竞争格局
- 看护城河

### 9.3 价值陷阱

**情况**：估值低，但逻辑变了

**案例**：康美药业
- 爆雷后PE跌到10倍
- 看起来便宜
- 但逻辑变了，价值毁灭

**应对**：
- 分析低估值的原因
- 判断是暂时还是永久
- 避开逻辑变坏的公司

### 9.4 历史估值陷阱

**误区**：
- 历史底部就是买点 → 逻辑可能变坏
- 历史顶部就是卖点 → 可能继续新高
- 简单对比历史 → 要考虑逻辑变化

**正确做法**：
- 先检查逻辑是否变化
- 逻辑不变才能参考历史
- 逻辑变了，历史估值没有意义

## 十、给初学者的相对估值速查表

1. **价值消费股**：先看PE
2. **成长白马**：先看PEG
3. **云SaaS**：先看PS
4. **银行券商**：先看PB
5. **保险**：先看PEV

---

## 十一、本章核心结论

- 相对估值好用，但前提是用对对象
- PE的核心决定因素是成长性、生命周期、商业模式和确定性
- PEG是成长股最实用的估值工具，PS适用于利润被前置投入压低但商业模式清晰的企业
- PB和PEV更适合高杠杆金融行业
- 相对估值一定要落到"预测财务指标 → 目标倍数 → 目标市值/目标价"的完整流程上
- 至少用两种方法交叉验证，避免单一方法的局限性
