# A股公司基本面分析技能

## 核心原则

本技能基于老裘（何老师）17年从业经验（10年行业分析师+7年基金经理）总结的方法论，用于系统化分析A股上市公司。

**为什么需要系统化方法？**
- 避免情绪化决策
- 排除财务造假风险
- 识别价值陷阱
- 找到真正的好公司

### 第一阶段：快速排雷（四不碰原则）

**立即执行，先于任何分析**

如果公司触犯以下任一原则，直接给出风险警告：

| 原则 | 检查方法 | 阈值 |
|------|----------|------|
| 大集团小公司 | 控股股东背景 | 集团收入>>上市公司收入 |
| 董事长想法太多 | 并购历史 | 频繁收购轻资产公司 |
| 杠杆率太高 | 资产负债率 | >70%直接不碰，>60%警惕 |
| 质押率太高 | 大股东质押比例 | >80%警惕，100%坚决不碰 |

**为什么要先排雷？**
- 研究好公司之前，先避开坏公司
- 节省时间，避免深度研究后才发现致命缺陷
- 保护本金，风险控制优先于收益追求

### 第三阶段：8个核心财务指标分析

按以下顺序计算和分析每个指标。

**数据获取**：优先使用妙想Skills获取数据
```bash
# 一次性获取多个指标
/mx-data "[公司名]营业收入、净利润、经营现金流、净资产、总资产、毛利率、ROE"
/mx-data "[公司名]最近8个季度营收 同比增长"
/mx-data "[公司名]商誉 净资产 资产负债率"
/mx-data "[公司名]研发费用 员工总数"
```

#### 指标1：现金收入比（最重要！）

```
现金收入比 = 销售商品提供劳务收到的现金 ÷ 营业收入 × 100%
```

**数据获取**：`/mx-data "[公司名]销售商品收到的现金 营业收入"`

| 判断标准 | 含义 |
|----------|------|
| ≥100% | 优秀，能拿到真金白银 |
| 80%-100% | 尚可，需关注应收账款 |
| <80% | 警告，收入质量存疑 |
| <50% | 高风险，可能存在收入注水 |

#### 指标2：净资产收益率(ROE)

```
ROE = 净利润 ÷ 净资产 × 100%
```

**数据获取**：`/mx-data "[公司名]近五年ROE 净资产收益率"`

| 判断标准 | 含义 |
|----------|------|
| 持续5年≥15% | 优秀公司 |
| 5%-15% | 普通水平 |
| <5%或为负 | 不推荐 |

#### 指标3：人均创收和人均创利

```
人均创收 = 营业收入 ÷ 员工总数
人均创利 = 净利润 ÷ 员工总数
```

**数据获取**：`/mx-data "[公司名]员工总数 营业收入 净利润"`

| 判断标准 | 含义 |
|----------|------|
| 人均创收 > 100万 | 良好 |
| 人均创利 > 10万 | 良好 |
| 人均创利 < 5万 | 需警惕 |

#### 指标4：总资产周转率

```
总资产周转率 = 营业收入 ÷ 总资产 × 100%
```

**数据获取**：`/mx-data "[公司名]总资产 营业收入"`

| 判断标准 | 含义 |
|----------|------|
| >100% | 优秀，资产利用率高 |
| 50%-100% | 正常 |
| <50% | 资产利用效率低 |

#### 指标5：季度收入增长率

```
季度收入增长率 = (本季度收入 - 去年同期收入) ÷ 去年同期收入 × 100%
```

| 判断标准 | 含义 |
|----------|------|
| 连续8季度>15% | 优秀 |
| 连续8季度>10% | 良好 |
| 增长率波动大或转负 | 需警惕 |

**数据获取**：`/mx-data "[公司名]最近8个季度营业收入 同比增长"`

**必须获取最近8个季度数据进行分析**

#### 指标6：商誉净资产比

```
商誉净资产比 = 商誉 ÷ 净资产 × 100%
```

| 判断标准 | 含义 |
|----------|------|
| <30% | 安全 |
| 30%-50% | 关注 |
| 50%-100% | 高风险 |
| >100% | 坚决不碰 |

**数据获取**：`/mx-data "[公司名]商誉 净资产"`

#### 指标7：毛利率

```
毛利率 = (营业收入 - 营业成本) ÷ 营业收入 × 100%
```

| 行业 | 优秀标准 |
|------|----------|
| 制造业 | >20% |
| 消费品 | >30% |
| 医药/白酒 | >50% |

**数据获取**：`/mx-data "[公司名]营业收入 营业成本 毛利率"`

#### 指标8：研发支出占比

```
研发支出占比 = 研发支出 ÷ 营业收入 × 100%
```

| 判断标准 | 含义 |
|----------|------|
| 10%-30% | 健康 |
| 20%左右 | 最佳 |
| <5% | 投入不足，缺乏创新 |
| >30% | 可能不可持续 |

**数据获取**：`/mx-data "[公司名]研发费用 营业收入"`

### 第四阶段：资产负债表深度分析

**为什么资产负债表最重要？**
- 包含信息最多
- 具有领先预判作用
- 利润表好看但资产负债表恶化 → 后续业绩会下滑

分析重点：
- 总资产变动（持续增长 vs 萎缩）
- 资产负债率（制造业>60%警惕，>70%不碰）
- 流动比率（合适值：2，<1流动性紧张）
- 资产/负债/权益结构

### 第五阶段：估值分析

#### 多种估值方法交叉验证

**方法1：历史估值法（看图法）**
- 查看公司历史PE/PB区间
- 当前估值处于什么位置
- 第一、二根线：高估；第三根线：合理；第四、五根线：低估

**方法2：相对估值法**
- PE估值：适用于盈利稳定公司
- PB估值：适用于金融、周期股
- PS估值：适用于亏损但成长公司

**方法3：分部估值法**
- 适用于多业务公司
- 各板块分别估值后加总
- 给予控股折扣（90%左右）

| 归母净利润 | vs去年同季度 | vs上一季度 | 连续8季度趋势 |
| 毛利率 | vs去年同季度 | vs上一季度 | 稳定性分析 |
| 净利率 | vs去年同季度 | vs上一季度 | 盈利能力变化 |

## 常见问题处理

**Q：公司数据缺失怎么办？**
A：明确标注"数据缺失"，跳过该指标，用其他指标综合判断。如果关键数据（如现金流、ROE）缺失，建议用户补充数据。

**Q：银行、保险、券商等金融行业怎么分析？**
A：金融行业属于特殊行业，分析方法不同：
- 资产负债率不适用（杠杆天然高）
- 估值主要用PB
- 关注行业特有指标（不良率、偿付能力等）

详见 `references/industry-standards.md` 中的"大金融行业"部分。

**Q：ST公司、退市公司怎么分析？**
A：ST公司需格外谨慎，原则上不建议普通投资者参与。退市公司不分析。

**Q：新股、次新股怎么分析？**
A：新股历史数据有限，重点关注招股说明书、行业前景、上市后业绩表现。

**Q：港股、美股中概股怎么分析？**
A：本技能主要针对A股，港股美股可参考但需注意会计准则差异。

## 执行步骤总结

### 每次分析必须遵循以下步骤：

**步骤1：数据收集**
- 优先使用妙想Skills（mx-data 和 mx-search）获取数据
- 使用 cn-financial MCP 作为备用数据源
- 确保获取所有必需的财务指标

**步骤2：执行分析**
- 按照八个阶段进行分析（排雷→行业→8指标→趋势→资产负债→估值→股价→打分）
- 根据公司所属行业应用相应标准
- 识别投资亮点和风险点

**步骤3：生成报告**
- 按照报告模板生成完整的 Markdown 报告
- 包含所有必需章节：总体评级、排雷检查、行业分析、8个指标、趋势分析、估值分析、投资建议

**步骤4：保存报告文件** ⚠️ **必须执行**

使用 Write 工具将完整报告保存为 MD 文件

**步骤5：输出摘要**
- 在对话中向用户简要说明分析结论

- 提供关键数据和建议摘要

## 分析原则总结

1. **数据优先**：用数据说话，不凭感觉
2. **趋势思维**：看长期趋势，不被短期波动迷惑
3. **对比思维**：与同行、与历史对比
4. **风险优先**：先排雷再淘金
5. **保持怀疑**：对异常数据保持警惕
6. **承认局限**：没有完美的分析方法，保持谦逊
