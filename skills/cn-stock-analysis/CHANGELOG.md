# cn-stock-analysis 技能更新日志

## v2.2 - 2026年4月8日（第三次更新）

### 新增功能

#### 同行业头部公司横向对比
- **位置**：行业分析章节的"行业横向对比分析"小节
- **功能**：与同行业头部公司进行多维度横向对比分析
- **对比维度**（6大维度，15个指标）：
  1. **规模对比**（权重15%）：总资产、营收、市值、员工数
  2. **盈利能力**（权重25%）：毛利率、净利率、ROE、人均创收、人均创利
  3. **成长能力**（权重20%）：营收增速、净利润增速、近3年CAGR
  4. **运营效率**（权重15%）：存货周转率、应收账款周转率、总资产周转率
  5. **财务健康**（权重10%）：资产负债率、流动比率、速动比率、现金占比
  6. **估值水平**（权重15%）：PE-TTM、PB、PS、估值分位

**对比公司选择标准**：
- 同行业内的头部公司（市值前3-5名）
- 主营业务相似度高
- 数据可获取且完整

**对比分析输出**：
- 6个维度的详细对比表格
- 每个表格包含：本公司、对比公司A/B/C、行业平均
- 综合评分与排名
- 对比分析结论（优势、劣势、行业地位、投资启示）

**数据来源标注**：每个对比表格都标注数据来源（cn-financial MCP工具）

## v2.1 - 2026年4月8日（第二次更新）

### 新增功能

#### 行业景气度分析
- **位置**：行业识别与分类章节的扩展
- **功能**：分析行业当前景气度和未来变化趋势
- **数据来源**：
  - cn-financial MCP工具：`get_industry_stocks`、`get_industry_list`、`get_sector_fund_flow`
  - 宏观数据：`get_macro_pmi`、`get_macro_cpi`、`get_bond_yield_curve`
  - 外部数据：国家统计局、工信部、行业协会

**五大评估维度**：
1. **供需关系**：需求端、供给端、库存水平
2. **价格趋势**：产品/服务价格、价格传导能力
3. **政策环境**：支持政策、限制政策
4. **资金动向**：板块资金流、机构持仓变化
5. **技术创新**：技术突破、产能替代

**景气度评级标准**：
- 高度景气（80-100分）：积极配置
- 景气（60-79分）：正常配置
- 平淡（40-59分）：谨慎配置
- 不景气（20-39分）：规避或减配
- 严重衰退（0-19分）：坚决回避

**各行业景气度分析要点**：
- 大消费：社零增速、消费者信心、季节性
- 大周期：产品价格、库存水平、周期判断
- 大金融：利率水平、社融规模、政策敏感度
- TMT：用户数、ARPU、研发支出、技术迭代
- 医药：审批数量、医保谈判、创新周期
- 高端制造：PMI、制造业投资、出口订单

**未来趋势判断三问**：
1. 行业处于周期的哪个阶段？
2. 行业未来3-5年的增长空间如何？
3. 行业面临的主要风险是什么？

### 报告模板更新

在"行业分析"章节中新增：
- **行业景气度分析**小节
- **供需关系**、**价格趋势**、**政策环境**、**资金动向**、**未来趋势判断**
- **景气度评估依据**表格

## v2.0 - 2026年4月8日（首次更新）

### 新增功能

#### 1. 同比/环比趋势分析
- **位置**：第五阶段（估值分析之前）
- **功能**：分析营收、净利润、盈利能力的同比和环比变化趋势
- **数据来源**：cn-financial MCP工具的`get_income_statement`和`get_growth_rates`
- **分析维度**：
  - 营收增长趋势（最近8个季度）
  - 净利润增长趋势（最近8个季度）
  - 盈利能力趋势（毛利率、净利率、ROE）
  - 趋势判断（加速/平稳/减速）

#### 2. 股价趋势分析
- **位置**：第七阶段（详细得分之前）
- **功能**：分析近期股价走势和技术指标
- **数据来源**：cn-financial MCP工具的`get_realtime_quote`和`get_historical_price`
- **分析维度**：
  - 当前股价状态（最新价、涨跌幅、年内涨跌）
  - 技术指标分析（MA均线、MACD、RSI、KDJ）
  - 支撑与压力位
  - 股价与基本面匹配度
  - 买卖时机判断

#### 3. 未来估值预估
- **位置**：估值分析章节的扩展部分
- **功能**：基于历史数据和合理假设进行估值预测
- **数据来源**：基于`get_valuation_metrics`和`get_income_statement`的历史数据
- **预估方法**：
  - PE/G估值法（适用于成长股）
  - PB估值法（适用于金融、周期股）
  - DCF估值法（适用于稳定盈利公司）
  - 三种情景分析（保守、中性、乐观）

#### 4. 数据来源验证
- **位置**：附录章节
- **功能**：明确标注所有数据的来源和验证方式
- **验证渠道**：
  - 上交所/深交所官网
  - 巨潮资讯网
  - cn-financial MCP工具（基于AKShare）
  - 第三方对比（东方财富、同花顺）

### 新增MCP工具说明

在cn-financial MCP工具使用指南中新增：

| 需求 | 工具 | 用途 |
|------|------|------|
| 季度利润表 | `get_income_statement` | 计算营收/净利润同比环比 |
| 季度增长率 | `get_growth_rates` | 直接获取增长率数据 |
| 历史估值 | `get_valuation_metrics` | 分析估值历史分位 |
| 日K线数据 | `get_historical_price` | 计算涨跌幅、技术指标 |
| 周线数据 | `get_historical_price` | 中期趋势分析 |
| 月线数据 | `get_historical_price` | 长期趋势分析 |

### 报告模板更新

新增章节：
- **四、同比/环比趋势分析**
- **七、股价趋势分析**
- **附录：数据来源与验证**

更新章节：
- **六、估值分析与未来预估**（原"估值分析"）

### 数据严谨性保证

1. **所有数据标注来源**：每个数据表都标注了数据来源
2. **计算公式公开**：关键指标的计算公式在技能说明中有明确描述
3. **多渠道验证**：提供官方验证渠道和第三方对比方法
4. **预测假设明确**：估值预估的假设条件和数据来源都有明确说明
5. **风险提示充分**：强调历史数据外推的局限性和投资风险

### 分析阶段调整

原分析流程：
1. 快速排雷 → 2. 行业识别 → 3. 8个指标 → 4. 资产负债表 → 5. 估值 → 6. 综合打分

新分析流程（v2.0）：
1. 快速排雷 → 2. 行业识别 → 3. 8个指标 → 4. 同比/环比趋势 → 5. 资产负债表 → 6. 估值与预估 → 7. 股价趋势 → 8. 综合打分

## v1.0 - 初始版本

### 核心功能

#### 基础分析框架
- **四不碰原则**：快速排雷检查
- **8个核心财务指标**：现金收入比、ROE、人均创收创利、资产周转率、季度增长率、商誉比、毛利率、研发占比
- **资产负债表分析**：1个金额、2个指标、3个结构
- **估值分析**：历史估值法、相对估值法、分部估值法
- **综合打分**：100分制评分系统

#### 数据获取策略
- cn-financial MCP工具优先
- 多期数据对比（至少5年）
- 季度数据用于增长率计算
