## 四、常见问题

### Q1：包装脚本仍然显示乱码

**解决方案**：
1. 确认使用 Python 3.6+
2. 不要直接在控制台查看 JSON 内容
3. 使用支持 UTF-8 的编辑器打开文件

### Q2：妙想Skills 调用失败

**检查项**：
1. mx-data 和 mx-search 路径是否正确
2. 路径中是否包含特殊字符
3. 是否有足够的权限执行脚本

### Q3：报告文件未生成

**检查项**：
1. SKILL.md 中是否包含强制要求
2. Write 工具是否正确调用
3. 文件保存路径是否有写权限
