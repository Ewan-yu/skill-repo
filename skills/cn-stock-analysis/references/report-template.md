# 分析报告输出模板

本文档包含完整的 A 股公司投资分析报告输出模板。分析完成后，严格按照此格式输出 Markdown 报告。

---

**所属行业**：[一级行业] - [二级行业]
**数据来源**：cn-financial MCP / 妙想 Skills / 巨潮资讯等

## 批量筛选场景模板

当用户提供一篮子股票要求筛选时：

### 1. 首先输出汇总对比表

```markdown
# 股票池筛选汇总

**分析日期**：YYYY-MM-DD
**筛选标的**：X只股票
**数据来源**：cn-financial MCP工具

## 汇总对比表

| 股票代码 | 股票名称 | 总分 | 评级 | 核心亮点 | 主要风险 |
|----------|----------|------|------|----------|----------|
| 600519 | 贵州茅台 | 95 | A+ | 现金流充沛、高ROE | 估值偏高 |
| 000333 | 美的集团 | 88 | A | 家电龙头、现金流好 | 增长放缓 |
| 002415 | 海康威视 | 75 | B+ | 安防龙头 | 政策风险 |
| ... | ... | ... | ... | ... | ... |
```

### 2. 按优先级输出详细报告

- **A级和A+级**：完整详细报告（使用完整模板）
- **B级**：简要报告（精简版）
- **C级**：仅说明风险，不生成详细报告

### 3. 最后给出筛选结论
