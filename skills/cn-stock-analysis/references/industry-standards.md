## 一级行业分类

| 行业类别 | 典型行业 | 核心特征 | 代表股票 |
|----------|----------|----------|----------|
| **大消费** | 白酒、食品饮料、日用品、家电、零售 | 高毛利率、品牌护城河、强现金流 | 贵州茅台、海天味业、美的集团 |
| **大周期** | 钢铁、煤炭、有色、化工、建材 | 周期性强、ROE波动大、价格敏感 | 中国宝安、江西铜业、海螺水泥 |
| **大金融** | 银行、保险、券商、信托 | 高杠杆、监管约束、PB估值为主 | 招商银行、中国平安、中信证券 |
| **TMT** | 电子、计算机、传媒、通信 | 高研发、技术更新快、用户价值 | 海康威视、立讯精密、科大讯飞 |
| **医药** | 化药、生物药、医疗器械、CRO | 高毛利、研发驱动、管线价值 | 恒瑞医药、药明康德、迈瑞医疗 |
| **高端制造** | 机械、电气设备、汽车零部件 | 适度研发、重资产、规模效应 | 三一重工、汇川技术、福耀玻璃 |

## 各行业详细评分权重

### 大消费行业

**评分权重配置**：

| 评估项 | 权重 | 说明 |
|--------|------|------|
| 现金收入比 | 10 | 消费终端现金流好 |
| ROE | 15 | 品牌溢价带来高ROE |
| 毛利率 | 15 | 消费品毛利率要求更高 |
| 增长能力 | 10 | 关注持续增长能力 |
| 商誉风险 | 10 | 警惕盲目并购 |
| 财务风险 | 10 | 资产负债率>60%警惕 |
| 研发创新 | 5 | 产品创新重要性 |
| 现金状况 | 5 | 现金充裕度 |
| 行业地位 | 15 | 品牌护城河深度 |
| 行业特有指标 | 5 | 品牌力、渠道覆盖 |
| **总分** | **100** | - |

**评分标准调整**：
- 毛利率：>30%优秀，>20%良好
- ROE：>18%优秀，>12%良好
- 现金收入比：>100%优秀

**行业特有指标**：
- 品牌力：市场地位、品牌认知度
- 渠道覆盖：线下渠道、电商渠道
- 产品结构：主力产品、新品贡献

**关键提示**：
- 关注品牌护城河深度
- 关注消费升级/降级趋势
- 关注新渠道影响（直播电商等）

### 大周期行业

**评分权重配置**：

| 评估项 | 权重 | 说明 |
|--------|------|------|
| 现金收入比 | 5 | 周期行业现金流波动大 |
| ROE | 10 | 看周期平均，不单看一年 |
| 毛利率 | 5 | 看趋势，不单看绝对值 |
| 增长能力 | 5 | 周期位置更重要 |
| 商誉风险 | 5 | 并购相对较少 |
| 财务风险 | 15 | 周期底部加杠杆很危险 |
| 研发创新 | 5 | 技术升级需求 |
| 现金状况 | 10 | 现金储备应对周期 |
| 行业地位 | 20 | 产能、成本优势 |
| 行业特有指标 | 20 | **周期位置、产能利用率** |
| **总分** | **100** | - |

**评分标准调整**：
- ROE：看周期平均，不单看一年（周期高点/低点差异大）
- 毛利率：看趋势，不单看绝对值（随周期波动）
- 资产负债率：>50%警惕

**行业特有指标**：
- 产能利用率：当前产能/总产能
- 产品价格：近期价格走势
- 周期位置：处于周期哪个阶段（复苏/繁荣/衰退/萧条）

**关键提示**：
- 周期股在周期底部往往是好买点
- 周期顶部高ROE可能是卖点
- 关注供给侧变化（限产、新增产能）

### 大金融行业

**评分权重配置**：

| 评估项 | 权重 | 说明 |
|--------|------|------|
| 现金收入比 | 5 | 金融业模式不同 |
| ROE | 15 | 监管约束上限 |
| 毛利率 | - | 不适用 |
| 增长能力 | 5 | 规模增长 |
| 商誉风险 | 5 | 并购相对较少 |
| 财务风险 | 20 | **风险控制核心** |
| 研发创新 | - | 不适用 |
| 现金状况 | 10 | 流动性管理 |
| 行业地位 | 15 | 市场份额 |
| 行业特有指标 | 20 | **不良率、偿付能力等** |
| **总分** | **100** | - |

**评分标准调整**：
- 资产负债率：不适用（金融业杠杆天然高）
- ROE：>12%优秀（资本充足率约束上限）
- 毛利率：不适用（金融业务模式不同）

**行业特有指标**：

**银行**：
- 不良贷款率：<1%优秀，<2%良好
- 拨备覆盖率：>150%优秀
- 净息差：>2%优秀
- 资本充足率：>11%满足监管

**保险**：
- 偿付能力充足率：>100%满足监管
- 新业务价值增长率：>10%优秀
- 内含价值增长率：>10%优秀
- 退保率：<5%良好

**券商**：
- 经纪业务收入占比：越低越多元化
- 投行业务排名：头部优势明显
- 自营业务收益：波动性大

**估值方法调整**：
- 主要用PB估值，PE为辅
- 关注净资产质量
- 关注监管政策变化

**关键提示**：
- 金融行业受监管严格
- 宏观经济影响大
- 关注利率环境变化

### TMT行业

**评分权重配置**：

| 评估项 | 权重 | 说明 |
|--------|------|------|
| 现金收入比 | 5 | SaaS公司现金流重要 |
| ROE | 10 | 盈利能力 |
| 毛利率 | 10 | 软件>50%，硬件20-30% |
| 增长能力 | 15 | **增长核心指标** |
| 商誉风险 | 5 | 并购较多 |
| 财务风险 | 5 | 轻资产，负债低 |
| 研发创新 | 15 | **技术创新核心** |
| 现金状况 | 5 | 现金储备 |
| 行业地位 | 15 | 市场份额、用户数 |
| 行业特有指标 | 15 | **用户价值、技术壁垒** |
| **总分** | **100** | - |

**评分标准调整**：
- 毛利率：>30%优秀（软件更高，硬件较低）
- 研发占比：>15%良好，>20%优秀
- 季度增长：看环比和同比（技术变化快）

**行业特有指标**：

**软件/互联网**：
- 用户数：DAU、MAU增长
- ARPU：每用户平均收入
- 付费率：付费用户占比
- 留存率：用户留存情况

**电子**：
- 技术壁垒：专利数量、核心技术
- 客户集中度：前五大客户占比
- 产业链地位：上游/中游/下游

**通信**：
- 5G建设进度：运营商资本开支
- 市场份额：运营商排名
- ARPU值：每用户收入

**关键提示**：
- 技术变化快，关注技术迭代
- 用户增长是关键指标
- 警惕技术颠覆风险

### 医药行业

**评分权重配置**：

| 评估项 | 权重 | 说明 |
|--------|------|------|
| 现金收入比 | 5 | 现金流重要性 |
| ROE | 10 | 盈利能力 |
| 毛利率 | 15 | 创新药毛利率高 |
| 增长能力 | 15 | **管线价值体现** |
| 商誉风险 | 5 | 并购较多 |
| 财务风险 | 5 | 负债通常较低 |
| 研发创新 | 20 | **创新驱动核心** |
| 现金状况 | 5 | 研发投入需要现金 |
| 行业地位 | 15 | 市场地位 |
| 行业特有指标 | 10 | **研发管线价值** |
| **总分** | **100** | - |

**评分标准调整**：
- 毛利率：>50%优秀（创新药更高）
- 研发占比：>10%良好，>15%优秀
- 季度增长：波动正常（受产品获批影响）

**行业特有指标**：

**研发管线**：
- 在研产品数量：管线丰富度
- 临床进展：各阶段产品数量
- NDA/BLA申请：上市申请数量

**监管进展**：
- NMPA审批情况：国内审批进度
- FDA审批情况：国际审批进度

**专利保护**：
- 核心专利到期时间：专利悬崖
- 专利布局：专利保护范围

**估值方法调整**：
- 创新药用DCF、Pipeline价值估值
- 关注研发管线价值
- 关注医保谈判影响

**关键提示**：
- 研发投入大、周期长
- 单一产品依赖风险
- 医保政策影响大

## 行业特有指标分析

### 大消费行业特有指标

#### 品牌力评估

| 维度 | 评估指标 | 数据来源 |
|------|----------|----------|
| 市场地位 | 行业排名、市场份额 | 行业报告、公司公告 |
| 品牌认知度 | 提及率、好评率 | 消费者调研 |
| 品牌溢价 | 价格vs竞品 | 电商平台价格对比 |

#### 渠道覆盖评估

| 渠道类型 | 评估要点 |
|----------|----------|
| 线下渠道 | 门店数量、覆盖区域、单店效益 |
| 电商渠道 | 天猫/京东/抖音旗舰店销量 |
| 新兴渠道 | 直播带货、社区团购 |

#### 产品结构评估

- 主力产品占比：Top 3产品营收占比
- 新品贡献：上市2年内产品营收占比
- 产品矩阵：高端/中端/低端产品分布

### 大周期行业特有指标

#### 产能利用率分析

```
产能利用率 = 当前产能 / 总产能 × 100%
```

| 产能利用率 | 行业状态 |
|-----------|----------|
| >90% | 供不应求，价格上行 |
| 70%-90% | 供需平衡 |
| <70% | 产能过剩，价格承压 |

#### 产品价格跟踪

| 数据类型 | 获取渠道 |
|----------|----------|
| 现货价格 | 期货交易所、大宗商品网 |
| 长单价格 | 公司公告、行业资讯 |
| 进出口价格 | 海关数据 |

#### 周期位置判断

| 周期阶段 | 特征 | 策略 |
|----------|------|------|
| 萧条期 | 价格跌破成本、产能出清 | 寻找机会 |
| 复苏期 | 需求回暖、价格企稳 | 积极配置 |
| 繁荣期 | 供不应求、价格上涨 | 谨慎持有 |
| 滞胀期 | 需求回落、产能过剩 | 逐步退出 |

### 大金融行业特有指标

#### 银行核心指标

| 指标 | 计算公式 | 判断标准 |
|------|----------|----------|
| 不良贷款率 | 不良贷款余额 / 总贷款 | <1%优秀，<2%良好 |
| 拨备覆盖率 | 贷款减值准备 / 不良贷款余额 | >150%优秀，>100%合格 |
| 净息差 | (利息收入-利息支出)/平均生息资产 | >2%优秀 |
| 资本充足率 | 资本净额 / 风险加权资产 | >11%满足监管 |

#### 保险核心指标

| 指标 | 说明 | 判断标准 |
|------|------|----------|
| 偿付能力充足率 | 实际资本 / 最低资本 | >100%满足监管 |
| 新业务价值 | 新单业务未来利润现值 | 增长率>10%优秀 |
| 内含价值 | 调整后净资产 + 有效业务价值 | 稳定增长 |
| 退保率 | 退保金 / 期初准备金 | <5%良好 |

#### 券商核心指标

| 指标 | 说明 |
|------|------|
| 经纪业务收入占比 | 越低说明业务越多元化 |
| 投行业务排名 | IPO承销数量、承销金额 |
| 自营业务收益 | 波动性大，关注风险控制 |

## 行业景气度分析

### 景气度评级标准

| 评级 | 分数 | 特征 | 策略 |
|------|------|------|------|
| **高度景气** | 80-100分 | 需求旺盛、价格上涨、政策支持、资金流入 | 积极配置 |
| **景气** | 60-79分 | 需求稳定、供需平衡、政策中性 | 正常配置 |
| **平淡** | 40-59分 | 需求一般、价格平稳、资金流出 | 谨慎配置 |
| **不景气** | 20-39分 | 需求疲软、价格下跌、政策限制 | 规避或减配 |
| **严重衰退** | 0-19分 | 行业危机、价格暴跌、政策打压 | 坚决回避 |

### 五大维度评估

| 维度 | 权重 | 核心问题 |
|------|------|----------|
| 供需关系 | 25% | 需求端状况、供给端产能、库存水平 |
| 价格趋势 | 20% | 产品价格变化、价格传导能力 |
| 政策环境 | 20% | 支持政策、限制政策 |
| 资金动向 | 15% | 板块资金流、机构持仓 |
| 未来增长 | 20% | 行业周期位置、3-5年增长空间 |

详细分析内容请参考主 SKILL.md 中的"行业景气度分析"章节。
