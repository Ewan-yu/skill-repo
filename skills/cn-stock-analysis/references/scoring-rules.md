## 评分体系概述

### 评分原则

**基础分：100分**

评分采用"扣分制为主，加分制为辅"的方式：
- 从100分开始
- 根据各项指标表现进行扣分或加分
- 最终得分对应相应评级

### 评分维度

| 评估维度 | 说明 |
|----------|------|
| 财务健康度 | 现金收入比、资产负债率、流动比率等 |
| 盈利能力 | ROE、毛利率、净利率等 |
| 成长能力 | 营收增长率、净利润增长率 |
| 运营效率 | 资产周转率、存货周转率等 |
| 创新能力 | 研发投入占比、研发成果 |
| 行业地位 | 市场份额、品牌影响力 |
| 风险控制 | 商誉风险、财务风险 |

## 通用评分规则

### 通用扣分项

**财务健康度扣分**：

| 项目 | 扣分 | 条件 | 说明 |
|------|------|------|------|
| 现金收入比<100% | -10分 | 连续3年不达标 | 收入质量存疑 |
| 现金收入比<80% | -15分 | 连续3年不达标 | 收入注水风险 |
| 现金收入比<50% | -20分 | 连续3年不达标 | 高风险 |
| 资产负债率>60% | -10分 | 制造业/消费业 | 财务风险增加 |
| 资产负债率>70% | -15分 | 制造业/消费业 | 高风险 |
| 资产负债率每超5% | -3分 | 在>60%基础上 | 累积扣分 |

**盈利能力扣分**：

| 项目 | 扣分 | 条件 | 说明 |
|------|------|------|------|
| ROE<15% | -10分 | 近3年平均 | 盈利能力一般 |
| ROE<10% | -15分 | 近3年平均 | 盈利能力弱 |
| ROE<5% | -20分 | 近3年平均 | 盈利能力差 |
| ROE下降趋势 | -5分 | 连续2年下降 | 趋势恶化 |
| 毛利率下降 | -5分 | 连续2年下降 | 竞争力下降 |

**成长能力扣分**：

| 项目 | 扣分 | 条件 | 说明 |
|------|------|------|------|
| 季度增长不达标 | -10分 | 未连续8季度>15% | 成长性不足 |
| 季度增长不达标 | -5分 | 未连续8季度>10% | 成长性一般 |
| 营收负增长 | -10分 | 当年度 | 业务萎缩 |
| 净利润负增长 | -10分 | 当年度 | 盈利下降 |

**风险控制扣分**：

| 项目 | 扣分 | 条件 | 说明 |
|------|------|------|------|
| 商誉净资产比>30% | -10分 | 当前值 | 减值风险 |
| 商誉净资产比>50% | -15分 | 当前值 | 高风险 |
| 商誉每超10% | -5分 | 在>30%基础上 | 累积扣分 |
| 研发投入<5% | -5分 | 科技公司 | 创新不足 |
| 研发投入<3% | -10分 | 科技公司 | 创新严重不足 |

### 通用加分项

| 项目 | 加分 | 条件 | 说明 |
|------|------|------|------|

## 评级标准

### 评分与评级对应

| 分数 | 评级 | 投资建议 | 特征 |
|------|------|----------|------|
| 90-100 | **A+** | 强烈推荐 | 优质公司，无明显短板 |
| 80-89 | **A** | 推荐 | 良好公司，少数关注点 |
| 70-79 | **B+** | 谨慎推荐 | 尚可公司，存在一些问题 |
| 60-69 | **B** | 持有/观望 | 一般公司，问题较多 |
| <60 | **C** | 不推荐 | 风险公司，不建议投资 |

### 各评级详细说明

#### A+ 级（90-100分）

**特征**：
- 所有核心指标优秀
- 无明显财务风险
- 行业龙头或领先者
- 现金流充沛

**适合投资者**：所有类型投资者

**操作建议**：可考虑长期持有

#### A 级（80-89分）

**特征**：
- 核心指标良好
- 财务风险可控
- 行业地位稳固
- 个别指标需关注

**适合投资者**：稳健型投资者

**操作建议**：适合中长期持有

#### B+ 级（70-79分）

**特征**：
- 部分指标良好
- 存在一些问题
- 行业地位一般
- 需要持续跟踪

**适合投资者**：有一定风险承受能力的投资者

**操作建议**：谨慎参与，密切关注

#### B 级（60-69分）

**特征**：
- 多项指标一般
- 问题较多
- 竞争力不足
- 不确定性大

**适合投资者**：投机型投资者

**操作建议**：观望为主，等待改善

#### C 级（<60分）

**特征**：
- 存在严重问题
- 财务风险高
- 竞争力弱
- 可能触犯一票否决项

**适合投资者**：不建议任何投资者

**操作建议**：坚决回避

## 一票否决项

触犯以下任一条件，直接评为 C 级（不推荐）：

| 否决项 | 说明 | 检查方法 |
|--------|------|----------|
| 审计非标意见 | 财务报表真实性存疑 | 查看审计报告 |
| 大股东质押率100% | 控制权风险 | 查看股东质押数据 |
| 连续亏损 | 持续经营能力存疑 | 查看利润表 |
| 重大违法违规 | 公司治理风险 | 查看公告、监管处罚 |
| 涉及财务造假 | 诚信风险 | 查看监管处罚、媒体曝光 |

## 特殊行业评分调整

### 金融行业评分调整

由于金融行业业务模式特殊，以下指标不适用或需调整：

- **资产负债率**：不适用（金融业杠杆天然高）
- **毛利率**：不适用（金融业务模式不同）
- **关注指标**：
  - 银行：不良贷款率、拨备覆盖率
  - 保险：偿付能力充足率、新业务价值
  - 券商：经纪业务收入、投行业务排名

### 周期行业评分调整

周期行业（钢铁、煤炭、有色、化工等）评分需考虑：

- **ROE**：看周期平均，不单看一年
- **毛利率**：看趋势，不单看绝对值
- **关注点**：周期位置、产品价格、产能利用率

### 新股/次新股评分调整

上市不足3年的公司：

- 历史数据有限，降低历史趋势权重
- 重点关注招股说明书、行业前景
- 关注上市后业绩是否变脸

## 评分使用指南

### 如何使用评分

1. **不要唯分数论**：分数是参考，不是绝对标准
2. **关注趋势**：分数的变化趋势比单次分数更重要
3. **结合估值**：好公司如果估值过高，也不是好买点
4. **考虑行业**：不同行业的评分标准不同
5. **持续跟踪**：定期更新评分，跟踪变化

### 评分局限性

1. **历史数据**：评分基于历史数据，不能预测未来
2. **行业变化**：行业环境变化可能影响评分有效性
3. **黑天鹅事件**：无法预测突发事件
4. **主观判断**：部分指标需要主观判断
5. **数据延迟**：财务数据存在更新延迟

### 评分最佳实践

1. **多维度验证**：结合技术分析、资金流向等
2. **与同行对比**：关注相对评分而非绝对评分
3. **关注变化**：评分变化比评分本身更重要
4. **长期跟踪**：建立评分历史，观察趋势
5. **动态调整**：根据公司实际情况调整权重
