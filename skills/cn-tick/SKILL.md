# A股实时行情与分时量能分析 v2.0

使用新浪财经接口获取A股数据，支持实时行情、分时K线、量能分析、同时段对比、承接力判断。

## 快速开始

### 实时行情

```bash
# 单只股票
uv run {baseDir}/scripts/analyze.py 600789

# 多只股票
uv run {baseDir}/scripts/analyze.py 600789 002446 002342

# 含分时量能分析（同时段对比+承接力判断）
uv run {baseDir}/scripts/analyze.py 600789 --minute

# 指定流通股数计算换手率（可从 cn-financial 获取）
uv run {baseDir}/scripts/analyze.py 600789 --minute --float-shares 1252270215

# JSON输出
uv run {baseDir}/scripts/analyze.py 600789 --json
```

### 持仓管理

```bash
# 显示持仓
uv run {baseDir}/scripts/portfolio.py show

# 添加持仓
uv run {baseDir}/scripts/portfolio.py add 600789 --cost 10.416 --qty 3400

# 更新持仓
uv run {baseDir}/scripts/portfolio.py update 600789 --cost 10.2

# 删除持仓
uv run {baseDir}/scripts/portfolio.py remove 600789

# 分析所有持仓（含分时量能）
uv run {baseDir}/scripts/portfolio.py analyze
```

## 股票代码格式

直接使用6位数字代码即可，系统自动识别市场：

- **沪市**: 6开头 (如 600789)
- **深市**: 0或3开头 (如 002446, 300001)
- **北交所**: 8或4开头

## 分时量能分析说明

### 成交分布时段

| 时段 | 说明 |
|------|------|
| 早盘30分 (9:30-10:00) | 主力早盘动作，>30%为抢筹信号 |
| 上午中段 (10:00-11:30) | 正常交易 |
| 下午中段 (13:00-14:30) | 正常交易 |
| 尾盘30分 (14:30-15:00) | >25%为尾盘异动信号 |

### 同时段量能对比（v2.0新增）

与前一交易日同时段对比，判断量能趋势：

| 量比范围 | 趋势判断 | 含义 |
|----------|----------|------|
| >1.3 | 明显增量 | 资金积极参与 |
| 1.1-1.3 | 温和增量 | 量能温和放大 |
| 0.9-1.1 | 基本持平 | 量能稳定 |
| 0.7-0.9 | 温和缩量 | 资金观望 |
| <0.7 | 明显缩量 | 资金离场或惜售 |

### 承接力判断（v2.0新增）

基于量价配合原则分析主力意图：

| 信号 | 条件 | 判断 |
|------|------|------|
| 缩量回调 | 价格下跌 + 量比<0.8 | 承接力强（主力护盘，散户惜售） |
| 放量下跌 | 价格下跌 + 量比>1.2 | 承接力弱（主力出货，恐慌抛售） |
| 放量上涨 | 价格上涨 + 量比>1.2 | 量价配合好（主力积极做多） |
| 缩量上涨 | 价格上涨 + 量比<0.8 | 量价背离（主力控盘或散户推动） |

### 早盘量能预期（v2.0新增）

基于早盘30分额比判断当日量能趋势：

| 额比 | 信号 | 图标 | 实战意义 |
|------|------|------|----------|
| >1.5 | 放量加速 | 🟢 | 主力强势介入，积极做多 |
| 1.2-1.5 | 温和放量 | 🟢 | 资金正常流入 |
| 0.8-1.2 | 量能持平 | 🟡 | 多空平衡 |
| 0.5-0.8 | 缩量明显 | 🟠 | 主力观望或控盘 |
| <0.5 | 极度缩量 | 🔴 | 资金离场或高度控盘 |

### 综合信号（v2.0新增）

系统自动汇总所有信号，用图标直观显示：

- 🟢 看多信号：承接力强、量价配合、早盘放量、主力抢筹
- 🟡 中性信号：量价背离、量能持平
- 🟠 警示信号：早盘缩量、承接力一般
- 🔴 看空信号：承接力弱、主力出货

### 换手率评级

通过 `--float-shares` 参数传入流通股数（可从 cn-financial 获取），计算换手率：

| 换手率 | 评级 | 说明 |
|--------|------|------|
| <1% | 极度低迷 | 无人问津 |
| 1-3% | 低迷 | 交投清淡 |
| 3-7% | 正常 | 常规交易 |
| 7-10% | 活跃 | 资金关注 |
| 10-15% | 高度活跃 | 热门标的 |
| >15% | 异常放量 | 筹码剧烈交换 |

### 主力动向信号

- **早盘放量 >30%**: 主力早盘抢筹
- **早盘放量 >40%**: 主力强势介入
- **尾盘放量 >15%**: 尾盘有一定放量
- **尾盘放量 >25%**: 可能有主力抢筹或出货
- **封板状态**: 涨停/跌停，关注封单量

### 放量时段 TOP 10

显示当日成交量最大的10个分钟，用于识别：
- 主力建仓/出货时机
- 重大消息发布时点
- 涨停封板/开板时刻

## 数据源

统一使用新浪财经接口，数据一致性好：

| 数据 | 接口 | 延迟 |
|------|------|------|
| 实时行情 | `hq.sinajs.cn` | ~3秒 |
| 分时K线 | `CN_MarketDataService.getKLineData` | ~1分钟 |

支持批量获取，减少请求次数。分时数据默认获取500条（覆盖2-3个交易日），支持同时段对比。

## 持仓文件

持仓数据存储在: `{baseDir}/data/portfolio.json`

## 限制

- 仅支持A股（沪深北交所）
- 交易时段外获取的是收盘数据
- 分时数据最多500条（约2-3个交易日）
- 换手率需要流通股数数据，可通过 cn-financial 获取
