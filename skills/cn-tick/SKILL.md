# A股实时行情与分时量能分析 v3.1

使用新浪财经接口获取A股数据，支持实时行情、分时K线、量能分析、同时段对比、承接力判断、VWAP、卖出三部曲。

## 快速开始

### 实时行情

```bash
# 单只股票
uv run {baseDir}/scripts/analyze.py 600789

# 多只股票
uv run {baseDir}/scripts/analyze.py 600789 002446 002342

# 含分时量能分析（同时段对比+承接力+三部曲卖出信号）
uv run {baseDir}/scripts/analyze.py 600789 --minute

# 指定流通股数计算换手率
uv run {baseDir}/scripts/analyze.py 600789 --minute --float-shares 1252270215

# JSON输出
uv run {baseDir}/scripts/analyze.py 600789 --json

# Markdown格式输出（适合导出文档）
uv run {baseDir}/scripts/analyze.py 600789 --minute --format md
```

### 持仓管理

```bash
# 显示持仓
uv run {baseDir}/scripts/portfolio.py show

# 添加持仓（支持 effective_float_shares 用于换手率计算）
uv run {baseDir}/scripts/portfolio.py add 600789 --cost 10.416 --qty 3400

# 更新持仓
uv run {baseDir}/scripts/portfolio.py update 600789 --cost 10.2

# 分析所有持仓（含分时量能+三部曲卖出信号）
uv run {baseDir}/scripts/portfolio.py analyze
```

## 股票代码格式

直接使用6位数字代码，系统自动识别市场：

- **沪市**: 6开头 (如 600789)
- **深市**: 0或3开头 (如 002446, 300001)
- **北交所**: 8或4开头
- **科创板**: 68开头 (±20%涨跌幅)
- **创业板**: 30开头 (±20%涨跌幅)

v3.1 基于实战资料全面审校，重点修复：

| 问题 | v2.0 | v3.1 |

| VWAP | 无 | ✅ 新增(黄线/均价线) |

## 分时量能分析说明

### 成交分布时段

| 时段 | 说明 |
|------|------|
| 早盘30分 (9:30-10:00) | 主力早盘动作，>30%为抢筹信号，<20%全天基本无戏 |
| 上午中段 (10:00-11:30) | 正常交易 |
| 下午中段 (13:00-14:30) | 正常交易 |
| 尾盘30分 (14:30-15:00) | >25%为尾盘异动信号 |

### 同时段量能对比（v3.1 以额比为主）

与前一/两交易日同时段对比，以额比为主判断量能趋势：

| 额比范围 | 趋势判断 | 含义 |
|----------|----------|------|
| >1.3 | 明显增量 | 资金积极参与 |
| 1.1-1.3 | 温和增量 | 量能温和放大 |
| 0.9-1.1 | 基本持平 | 量能稳定 |
| 0.7-0.9 | 温和缩量 | 资金观望 |
| <0.7 | 明显缩量 | 资金离场或惜售 |

额比比量比更重要：价格上涨时成交额同步放大，额比能更真实反映资金参与度。

### 承接力判断（v3.1 基于昨收价）

基于昨收价（而非当日第一根K线）判断量价配合：

| 信号 | 条件 | 判断 |
|------|------|------|
| 缩量回调 | 跌>0.5% + 额比<0.8 | 承接力强（主力护盘，散户惜售） |
| 放量下跌 | 跌>0.5% + 额比>1.2 | 承接力弱（主力出货，恐慌抛售） |
| 放量上涨 | 涨>0.5% + 额比>1.2 | 量价配合好（主力积极做多） |
| 缩量上涨 | 涨>0.5% + 额比<0.8 | 量价背离（主力控盘或散户推动） |

### VWAP（均价线）

成交量加权均价（黄线），是实战中最重要的参考线之一：

- **价格在均价线上方**：多头占优
- **价格跌破均价线**：卖出信号（三部曲核心规则）
- **价格高出均价线>3%**：先卖部分锁利

### 卖出三部曲信号（v3.1 新增）

基于实战资料的核心规则，每只持仓股自动检测：

| 持仓类型 | 卖出条件 |
|---------|---------|
| 前日涨停股 | 开盘30分钟不红盘(低于昨收) → 卖出 |
| 前日未涨停股 | 开盘1小时不红盘 → 卖出 |
| 冲高回落 | 盘中涨幅超2%后回落破均价线 → 卖出 |
| 尾盘 | 不涨停 → 卖出 |

### 换手率评级（v3.1 对齐实战标准）

| 换手率 | 评级 | 说明 |
|--------|------|------|
| <1% | 极度低迷 | 无人问津 |
| 1-3% | 低迷 | 交投清淡 |
| 3-7% | 正常 | 常规交易 |
| 7-15% | 活跃 | 资金关注 |
| 15-25% | 高度活跃 | 热门标的 |
| >25% | 异常放量 | 筹码剧烈交换，需警惕 |

注：大盘蓝筹和小盘题材股的活跃阈值差异大，具体需结合市值判断。

### 实时量比

从新浪实时数据提取（字段30），计算公式：当日累计成交量 / 过去5日平均每分钟成交量 × 累计分钟数。

| 量比 | 含义 |
|------|------|
| >2.0 | 明显放量，资金活跃 |
| 1.5-2.0 | 温和放量 |
| 0.8-1.5 | 正常 |
| <0.8 | 缩量，资金观望 |

### 涨停/跌停检测（v3.1 修复）

基于昨收价正确计算：
- 普通股: 昨收 × 1.10（涨停）/ × 0.90（跌停）
- ST股: 昨收 × 1.05 / × 0.95
- 科创/创业板: 昨收 × 1.20 / × 0.80

### 放量时段 TOP 10

显示当日成交量最大的10个分钟，用于识别主力建仓/出货时机。

## 数据源

统一使用新浪财经接口：

| 数据 | 接口 | 延迟 |
|------|------|------|
| 实时行情 | `hq.sinajs.cn` | ~3秒 |
| 分时K线 | `CN_MarketDataService.getKLineData` | ~1分钟 |
| 实时量比 | 新浪实时数据字段30 | ~3秒 |

## 数据时效

系统自动标注当前市场状态：
- 交易中(上午/下午)：实时数据，约3秒延迟
- 盘前/午间休市/已收盘/休市(周末)：快照数据，非实时

## 持仓文件

持仓数据存储在: `{baseDir}/data/portfolio.json`

持仓格式支持 `effective_float_shares` 字段（用于换手率计算）。

## 限制

- 仅支持A股（沪深北交所）
- 交易时段外获取的是收盘数据（有标注）
- 分时数据最多500条（约2-3个交易日）
- 换手率需要流通股数数据，可通过 cn-financial 获取
