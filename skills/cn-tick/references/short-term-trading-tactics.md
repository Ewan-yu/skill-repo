# 短线实战技巧参考手册

> 来源：游资实战笔记整理。适用于 A 股超短线（T+1）操作，供 cn-tick 分析时参考。

---

## 一、预期与预期差

### 什么是预期

市场对个股次日走势有一个普遍预期。预期来自前一天的盘面表现：

| 前一天表现 | 市场预期 |
|---|---|
| 强势大封单涨停 | 次日高开，甚至顶一字 |
| 大热门 / 重大利好 | 次日竞价封单 10 亿以上 |
| 连续跌停 | 次日继续跌停 |

### 低于预期 vs 高于预期

- **低于预期**：强势涨停，次日却低开；预期顶一字，结果竞价只有 1 亿封单。
- **高于预期**：预期跌停，结果只跌 5 个点；预期封单 10 亿，结果来了 50 亿。

### 预期差的使用原则

1. **先有预期，才有预期差**。不能把所有低开都当预期差。
2. 预期差要和**大盘共振**配合——大盘明显普涨时，近期活跃股低位的预期差才有效。
3. 板块跟风不够、个股量能不够，即使高于预期也可能坑人。
4. 预期差是博弈工具，不是无脑信号。

## 二、量能分析

### 核心原则

> 量是价的先行指标。量能不足，价格涨不起来。

### 判断缩量 / 放量的方法

**对比同期量能**（最实用方法）：

- 对比前 1-2 天**同一时段**的成交量。
- 强势股早盘半小时的量应占全天量的 **30% 以上**。
- 早盘半小时量只有全天的 **20%** → 除非突发利好，当天基本没戏。
- 成交超 20 亿的股越容易看出趋势；小票（小盘股）不适用，一个大单就能改变趋势。

**集合竞价量的查看（通达信）**：

1. 光标移到目标日期，回车打开当天分时图。
2. 按 `Ctrl+W` 打开分时区间统计窗口。
3. 选择"大单成交"，按时间排序，查看 9:25 的成交量。
4. 成交金额 = 成交手数 × 100 × 开盘价。

**竞价量对比应用**：
- 今日竞价量 < 昨日竞价量的 70% → 缩量，不好做 T，冲高概率低。
- 今日竞价量 > 昨日竞价量 → 放量，有承接基础。

### 量比的使用

- 量比 = 当前每分钟成交量 / 过去 5 日平均每分钟成交量。
- 前 4 天换手率低（1% 以下），今天量比突然放到 10 倍 → 重点关注。
- 前 4 天换手率已在高位且涨幅不小，今天量比 2 以上 → 警惕出货。
- 牛市一般在量比 **2-3** 中找涨停股，量比太高反而不好。

## 三、量价关系与承接力

### 什么是承接力

承接力 = **真实成交 + 股价维持不跌**。

> 买卖盘中的托单不是承接力，托单是庄家的迷惑性动作，快要成交时会撤单。

### 分时图判断承接力

白线 = 实时股价；黄线 = 当前所有成交量的均价（VWAP）。

| 形态 | 含义 | 操作参考 |
|---|---|---|
| 黄线横盘，白线沿黄线上下小幅震荡 | 量到位后即将拉升 | 等量能到位后买入 |
| 黄线横盘或微向上，白线在黄线上方 | 庄家控盘高，多出现在低位 | 庄股特征，底部可介入 |
| 黄线向上，白线沿黄线上走 | 当天 1 小时量超前一天全天量的 60% | 强势信号，可跟进 |
| 黄线横盘，白线在黄线下方较远后反弹 | 转势信号 | 激进在 A 点买，稳妥在突破黄线后 B 点买 |

### 量能承接的量化参考

- 半小时成交量 ≥ 前一天全天成交量 → 强承接，大概率拉升。
- 1 小时成交量 ≥ 前一天全天成交量的 60% → 承接力良好。
- 同期量能低于前一天同期 50% → 缩量，承接不足。

## 四、支撑位判断

### 简单支撑位

- **5 日线**：最常用的短线支撑位。5 日线上持有，5 日线下走。
- **黄线（VWAP）**：日内支撑，白线跌破黄线且无法收复 → 弱势信号。
- **前期高点 / 低点**：次要参考。

### 支撑位结合量能判断守仓

守仓三要素（同时满足才守）：

1. 分时黄线平稳或向上（不向下）。
2. 离下一个支撑位不远（5 日线附近）。
3. 没有明显缩量（量能维持同期水平）。

> 缩量 + 分时向下 + 离支撑位远 = 早走早好，不要犹豫。

## 五、卖出三部曲

> 适用于：已持仓，决定某天要卖出时，判断最佳卖出时机。

### 三个卖出点（按优先级）

**第一部曲：开盘后不红盘卖出**

- 前一天涨停的股：开盘后 **半小时** 不翻红 → 卖出。
- 前一天未涨停的股：开盘后 **1 小时** 不翻红 → 卖出。
- 例外：开盘 -4 个点但分时图持续向上爬、黄线向上 → 可适当等待。

**第二部曲：冲高回落卖出**

- 冲高超过 2 个点，回落破黄线 → 卖出。
- 盘中涨停后打开涨停板 → 打开瞬间卖出。
- 为避免卖飞：白线离黄线 3 个点以上时先卖出一部分锁定利润。

**第三部曲：尾盘不涨停卖出**

- 前两条均未触发，坚持到尾盘。
- 尾盘未涨停 → 卖出。

### 特殊情况

| 情况 | 处理 |
|---|---|
| 庄股 / 妖股 | 坚持到尾盘，但必须是在 K 线底部介入，涨了 40-50% 后不适用 |
| 大利空 / 盘面风向突变 | 个股快跌停时（-9 个点）止损 |
| 成交量超 50 亿的趋势大票 | 持有到尾盘，破 5 日线才走 |

## 六、买入时机与低吸原则

### 低吸三原则

1. **必须是近期热点**：不对没热过的板块做预判，不买冷门股。
2. **低吸不追高**：介入不超过 3 个点，最好绿盘介入，分时白线在黄线以上。
3. **超短就是超短**：次日不强势涨停必须出局，不做成短线中线。

### 买入信号组合（三个最简单条件）

1. K 线在 5 日线上，最好在 60 日线上。
2. 分时白线在黄线之上。
3. 符合近期热点。

### 买入价格控制

- 一般情况：买入不超过 3 个点（相对昨收）。
- 特别强势股：不超过 6 个点。
- 理由：控制次日低开的风险敞口。

## 七、缩量信号与操作

### 缩量的判断

- 对比前 1-2 天同期量能（同一时段）。
- 早盘半小时量 < 全天量的 20% → 缩量，当天基本没戏。
- 15 分钟 K 线上看量柱明显低于前几天同期 → 缩量确认。

### 缩量时的操作逻辑

```
缩量 + 分时黄线向下 + 离支撑位远 → 立即卖出，不等
缩量 + 分时黄线平稳 + 靠近支撑位 → 可以等，观察是否企稳
缩量 + 分时黄线向上 → 暂时观察，可能是洗盘
```

## 八、涨停与连板分析

### 涨停股的量能特征

- 快速封板（一笔拉板）：主力占总金额 30% 以上，散户参与少。
- 正常涨停：主力买入占总金额 8-15%。
- 封单金额 > 流通市值 5% → 封板较稳。

### 连板股的操作逻辑

- 情绪连板涨停股：次日半小时不红盘走（三部曲第一条）。
- 妖股 / 庄股：次日留到尾盘，不涨停再走。
- 龙头断板后：次日谨慎，跌停数增加时不出手。

### 涨停打开的处理

- 涨停打开 → 打开瞬间卖出（三部曲第二条）。
- 打开后快速重新封板 → 可以重新买入（强势信号）。

## 九、龙虎榜实战解读

### 快速判断逻辑

| 龙虎榜特征 | 次日预判 |
|---|---|
| 敢死队占 2 席且买入超总额 1/3 | 次日高开砸盘，高开不追 |
| 买 1 独大（是买 2-5 的几倍） | 吃独食，低开可进，高开当心 |
| 买 1-5 金额平均，无敢死队 | 相对安全，有肉吃 |
| 卖出比买入大 50% 以上 | 小心次日继续下跌 |
| 冷门营业部出现 | 可能举牌 / 利好泄露 / 私募建仓 |
| 山东帮出没 | 妖股行情，走后暴跌，做好接跌停准备 |

### 市场口诀

> 强势看金田，转折看淮海，堵单看佛山，弱势看校长（成都帮）。

- 金田路频繁出没 → 市场热情高，连板概率高，行情好做。
- 成都帮（校长）出没 → 市场弱势。

## 十、仓位与风控

### 仓位控制原则

- 短线资金只占总资金的 **30%**。
- 这 30% 只用 **一半** 买入，保持机动灵活性。
- 不满仓操作，避免被迫卖出好股去买新股。
- 月初把上月盈利提取出来，保证本金安全。

### 个股对冲

- 每次买入 4-6 只**不同板块**个股（如 2 只热点 A + 1 只热点 B + 1 只热点 C）。
- 存量资金博弈下，一个板块跌了另一个可能涨，控制整体胜率。

### 止损纪律

- 超短线：次日不强势涨停必须出局，不拖。
- 趋势股：破 5 日线走，不破不走。
- 极端情况（大利空开盘快速跌停）：-9 个点止损。

## 十一、盘中看盘要点

### 分时图核心观察

1. **白线 vs 黄线**：白线在黄线上方 = 多头占优；白线跌破黄线且无法收复 = 弱势。
2. **黄线方向**：黄线向上 = 均价成本上移，买方主动；黄线向下 = 卖方主动。
3. **量柱节奏**：早盘放量 + 黄线向上 = 主力建仓；尾盘放量 + 黄线向下 = 主力出货。

### 主力监控信号

- 主力监控全绿 → 个股先于指数下跌，立即卖出。
- 主力监控突然大量变红 → 市场开始反弹，可买入。

### 换手率的实际计算

> 名义换手率需修正：扣除前 10 大流通股东持仓后，才是真实可交易换手率。

- 例：换手率 5%，前 10 大股东持有 80% 流通股 → 真实换手率 = 5% ÷ 20% = **25%**。
- 热门股上涨真实换手率 20% 以上；次新股上涨期间 40% 以上。

## 十二、交易心态与纪律

### 超短线最重要的一条：纪律

- 快进快出，不贪，不被套等解套。
- 注定会卖飞，注定会错过，这是必然的代价。
- 从大概率角度看，纪律执行是对的。

### 复盘判断逻辑

| 复盘判断 | 次日实际 | 操作 |
|---|---|---|
| 判断涨 | 开盘涨 | 果断买入 |
| 判断涨 | 开盘跌 | 放弃 |
| 判断跌 | 开盘涨 | 立马承认错误，介入 |
| 判断跌 | 开盘跌 | 不关注 |

### 心态要点

- 能发现强势股但不敢上 = 心理素质问题，不是技术问题。
- 不要借钱炒股，建议不超过存款的一半炒股。
- 每天盈利 2-3% 就满足，月盈利 15-16%，靠复利积累。
- 保证本金安全是第一位，安全性优先于盈利性。

## 附：关键指标速查

| 指标 | 强势信号 | 弱势信号 |
|---|---|---|
| 早盘半小时量占比 | ≥ 30% | ≤ 20% |
| 竞价量 vs 昨日竞价量 | ≥ 100% | ≤ 70% |
| 量比 | 2-3（牛市找涨停） | > 5 且高位（警惕出货） |
| 白线 vs 黄线 | 白线在黄线上方 | 白线跌破黄线 |
| 黄线方向 | 向上或横盘 | 向下 |
| 1 小时量 vs 昨日全天量 | ≥ 60% | ≤ 30% |
| 封单金额 | > 流通市值 5% | 封单持续减少 |
